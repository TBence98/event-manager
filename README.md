# Event Manager

## Getting started

```bash
npm install
```

This project uses a custom development build (`expo-dev-client`), not Expo Go. You need to build and install the native app before you can run it.

**First time (or after native dependency changes):**

```bash
npm run ios      # iOS simulator or device
npm run android  # Android emulator or device
```

**Day-to-day development (after the app is already installed):**

```bash
npm start
```

Then open the installed dev client on your simulator or device. It will connect to the Metro bundler automatically.

For web:

```bash
npm run web
```
